## Http Cron Job
> Chama uma rota http dentro de um tempo agendado.

### Como executar
> O projeto deve estar na Gopath
```sh
cd $GOPATH/src/github.com/mrbrunelli/rotina-portal-fatura
```
> Execute o sh script.
```sh
$ ./run.sh
```
